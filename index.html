📋 Règles — Ultimate TTT

Structure

Le plateau est une grille 3×3 de sous-grilles, chacune étant elle-même un morpion 3×3 classique.

Comment jouer

La case choisie dans une sous-grille détermine la sous-grille où l'adversaire doit jouer. Ex : position centrale → sous-grille centrale.

Grille bloquée

Si la sous-grille imposée est déjà gagnée ou pleine, le joueur peut jouer dans n'importe quelle grille active.

Auto-complétion

Si un coup gagne la sous-grille active, il est joué automatiquement. Pas le choix !

Victoire

Aligner 3 sous-grilles gagnées en ligne, colonne ou diagonale.
